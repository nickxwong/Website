Like most of the projects I work on, Workout Creator started out as something for personal use. Lifting weights is a big hobby of mine and I frequently found myself putting together different lifting plans on Excel. However, things like spreadsheet formatting or coming up with automated formulas to track the distribution of exercises to muscle groups would end up taking almost as much time as planning the workouts themselves, so I decided to create a full stack web application to abstract away the tedious, behind-the-scenes parts.

On the front-end, I used HTML, CSS, Javascript, and the React framework. The back-end was developed with a combination of Express, Node.js, and JavaScript. For persistent storage of data, such as exercises, user accounts, and saved workouts, a MySQL database was used. The entire application is deployed on Heroku.

Project walkthrough

Exercise List

To create their workouts, users can select from a list of exercises. Each exercise in the system has one to three target muscle groups, which are listed below the exercise's name. Users can select a muscle group to filter the list with, allowing for the easy creation of body part-centric workouts, like a leg day or a chest day.

The exercise list, which is populated from a table on a MySQL database, consists of premade exercises that will appear for every user, as well as custom exercises that can be created by signed-in users.

Exercise Creator

Custom exercises are linked to user accounts, so they can only be seen and used by the user who created them.

Workout

Once exercises are selected, the workout can be further customized by adding sets and rep ranges to each exercise. Exercises can also be rearranged by dragging and dropping rows into the desired positions. When hovering over each row with a mouse, a trash icon appears that users can use to remove an exercise if needed.

The distribution of sets per muscle group is tracked automatically and displayed below the workout. This will make it easier for users to determine if their workouts are properly balanced.

Above the workout, users can name their workouts and save them to their profiles. Saved workouts will be listed in the user dashboard.
